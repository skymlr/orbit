# Quick Capture Override

## Intent

This is Orbit's fastest surface. It should feel lightweight, keyboard-first, and transient.

## Rules

- The panel keeps the master `14pt` outer radius with dark glass at `0.32`.
- The editor well uses the `8pt` radius and `ultraThinMaterial`.
- Markdown formatting controls should stay visually quiet so the editor remains primary.
- Category chips can wrap aggressively; the composer must still read as a single compact unit.
- The save action remains the strongest control in the bottom row.
- Shortcut hints should stay visible on preview toggle and save affordances.

## Avoid

- Do not add decorative illustrations or large empty-state blocks.
- Do not add multi-step flows or modal detours inside quick capture.
