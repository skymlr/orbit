# Workspace Settings Override

## Intent

Settings should share the same cosmic shell but recede behind clarity. This area is configuration-first, not theatrical.

## Rules

- Sidebar remains functional and understated; detail content carries the visual weight.
- Setting groups use the master spacing rhythm and center on the `700pt` section width.
- Buttons should retain Orbit styling, but fields and labels should stay native and readable.
- Category color picking is one of the few places where a denser visual grid is correct.
- About content should read as structured product information, not marketing copy.

## Avoid

- Do not add large hero banners or oversized illustrations.
- Do not replace standard text fields with stylized custom controls unless there is a real workflow need.
