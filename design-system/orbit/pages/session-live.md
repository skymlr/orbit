# Session Live Override

## Intent

The live session view is the operational console. It should feel focused, editable, and slightly more serious than the menu surfaces.

## Rules

- Content column stays capped near `700pt` inside the wider workspace frame.
- Session title is the strongest typographic element on the page.
- Task filter controls should stay above the task list and remain low-friction.
- Task rows own the visual rhythm; new supporting controls should not visually overpower them.
- Empty states remain glass cards with concise guidance and shortcut reinforcement.
- Toolbar actions stay icon-led and lightweight.

## Avoid

- Do not add dashboard-style metric tiles above the task list.
- Do not fragment the page into many competing cards.
